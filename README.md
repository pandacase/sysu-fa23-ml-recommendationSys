# Build & Run

1. run all in main.ipynb


2. in the `frontend/`:

    ```bash
    npm install
    npm run build
    ```

3. `pip install streamlit` 

    recommended python version: 3.8 with brand new enviroment

4. run server:

    ```bash
    streamlit run app.py
    ```
